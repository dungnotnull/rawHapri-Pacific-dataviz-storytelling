# Data sources

| File | Source | Notes |
|---|---|---|
| `sea_level.json`, `sea_level_pacific_avg.json` | Pacific Data Hub, dataset `CLIMATE_CHANGE_SEA_INDICATORS` (`all_data_v1.xlsx`), accessed 2026-07-07 | Meters, relative sea-level change, 21 PICTs, 1993–2023. `_pacific_avg` is an **unweighted** mean across reporting PICTs per year - a simple cross-country average for a headline line, not a tide-gauge-weighted regional index. |
| `ghg_per_capita.json` | Pacific Data Hub, dataset `CLIMATE_CHANGE_GHG_INDICATORS` (`all_data_v1.xlsx`), accessed 2026-07-07 | Tons CO2 per capita, 17 PICTs, 1970–2024. Palau (86.7 t) and New Caledonia (18.4 t) are genuine reported outliers - Palau's figure reflects its large international ship registry / bunker fuel accounting, New Caledonia's reflects nickel-processing industry, not typical household consumption. Flag both in the UI rather than hiding them. |
| `temperature_anomaly.json`, `temperature_pacific_avg.json` | Pacific Data Hub, dataset `CLIMATE_CHANGE_TEMP_INDICATORS` (`all_data_v1.xlsx`), accessed 2026-07-07 | Degrees Celsius anomaly vs. baseline, 22 PICTs, 1850–2025. `_pacific_avg` is an unweighted cross-country mean per year, same caveat as sea level. |
| `emitters_context.json` | Global Carbon Project / Our World in Data, CO2 emissions per capita, 2023 (approximate, retrieved via web search 2026-07-07); Pacific figure computed from `ghg_per_capita.json` (unweighted mean of 15 PICTs, excluding Palau and New Caledonia outliers) | **Not part of the official Pacific Data Hub extract** - used only as outside context to size the PICs-vs-major-emitters gap. Figures are approximate/illustrative; replace with the team's own preferred comparator set if desired. |
| `migration_placeholder.json` | **Fully placeholder** - the team has not finalized migration analysis | Loosely modeled on the *shape* of data available on the IOM Asia-Pacific Migration Data Portal (https://ap-migrationdata.iom.int/en), but every number is invented and must be replaced before submission. Do not cite this file's figures as real in the final entry. |
| `countries-50m.json` | `world-atlas` npm package (Natural Earth 1:50m, public domain) | Basemap land/country outlines only, used for geographic context on the Cause section map. |
